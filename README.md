# 图像缩小、恢复与频域分析实验
## 实验目的
掌握图像下采样与恢复的基本方法，理解不同插值算法对图像质量的影响，并通过傅里叶变换和DCT变换分析图像在频域中的特性。

## 实验环境
Python 3.x
OpenCV
NumPy
Matplotlib
SciPy
Tkinter

## 实验步骤
1. 图像读入
通过文件选择对话框读入灰度图像，若图像为彩色则自动转换为灰度图。
2. 下采样
将原始图像缩小为原来的1/2，使用最近邻插值实现（无预滤波）。
3. 图像恢复
分别使用三种插值方法将缩小图恢复至原始尺寸：
最近邻插值
双线性插值
三次内插值
4. 空间域评价
显示原图、缩小图、三种恢复结果
计算每种恢复方法的MSE和PSNR
5. 傅里叶变换分析
对原图、缩小图、双线性恢复图分别计算二维FFT：
将频谱中心移动到图像中心
对幅度谱取对数显示
比较高频成分差异
6. DCT分析
对原图和三种恢复图分别进行8×8分块DCT：
每块只保留6个最低频系数（其余置零）
显示DCT系数图
显示重建图像
计算保留能量占比

## 实验结果
空间域评价：
三种恢复方法的MSE和PSNR对比显示，双线性插值和三次内插值明显优于最近邻插值，其中双线性插值在平滑度和清晰度之间取得较好平衡。
傅里叶频谱分析：
频谱图中心最亮对应低频信息，四周较暗对应高频细节。下采样后高频丢失或混叠，频谱四周变暗；双线性恢复进一步抑制高频，能量更向中心集中。
DCT分析：
8×8分块DCT后仅保留6个低频系数，重建图像仍可识别，说明能量集中在低频区域。不同恢复方法的保留能量占比差异反映了各自引入的伪影或平滑程度。
